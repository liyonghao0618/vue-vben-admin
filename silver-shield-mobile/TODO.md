# Silver Shield Mobile Apps TODO

# 先用H5开发和调试,最终进度达到95%的时候移动到微信小程序进行测试

## 目标

为 `silver-shield-mobile-apps` 落地一个双用户端移动应用项目，支撑：

- 老年用户端
- 子女 / 守护人端
- 手机聊天能力
- App 内语音通话能力
- 接入 `ai-api-service` 的诈骗识别能力
- 与主业务系统的绑定关系、通知、求助、风险记录联动

## 当前建议路线

当前最推荐的落地路径：

1. 先做“一个应用双角色”的 `uni-app` MVP
2. 先支持文本聊天和风险识别
3. 再接 App 内语音通话
4. 再补更完整的通话后摘要分析
5. 最后再评估是否拆成两个独立应用
