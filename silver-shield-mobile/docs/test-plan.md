# 测试计划

## 当前已具备可测试范围

### 登录与角色

- 老年端登录、进入首页、退出登录
- 守护人端登录、进入首页、退出登录
- 登录失败提示与角色切换

### 聊天与风险识别

- 文本消息发送与接收
- 链接识别
- 图片消息 OCR 预留识别
- 风险列表刷新
- 风险详情回访与远程提醒

### 求助与社区协同

- 老年端一键求助
- 守护人端查看求助详情
- 标记处理中
- 触发社区协同入口

### 通话与复盘

- 拨打、接听、挂断、异常
- 通话记录沉淀
- 通话摘要保存
- 通话后风险复盘提醒

### 适老化

- 大字号
- 高对比
- 简化模式
- 语音播报预留提示

## 建议执行方式

1. `npm run dev:h5:demo` 做演示回归
2. `npm run dev:h5` 做联调回归
3. `npm run type-check`
4. `npm run build:h5`
