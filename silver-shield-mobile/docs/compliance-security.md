# 合规与安全说明

## 隐私政策范围

当前产品需要覆盖的数据范围：

- 登录账号与手机号
- 绑定关系信息
- 聊天消息与风险识别结果
- 求助记录与处置状态
- 通话记录与后续摘要

## 用户授权弹窗内容

建议在小程序 / App 阶段明确提示：

- 麦克风权限：用于语音通话
- 通知权限：用于风险提醒、求助提醒、远程提醒
- 位置权限：仅在发起求助并明确授权时使用

## 敏感权限最小化申请

- H5 阶段不主动申请额外高敏权限
- 语音能力落地前不申请麦克风
- 求助定位未开启前不申请位置权限

## 通信数据展示脱敏策略

- 手机号仅展示前 3 后 4
- 风险详情中对链接、卡号、验证码做遮罩
- 日志、埋点、错误上报中禁止上传完整消息正文

## 风险数据访问权限控制

- 老人端仅查看与自己相关的风险提示
- 守护人端仅查看绑定老人数据
- 高风险详情与求助详情必须依赖登录态
- 退出登录后清空本地 token 和会话上下文
