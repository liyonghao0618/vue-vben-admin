# Silver Shield Mobile 手工测试清单

## 登录与角色

- [ ] 老年端账号可成功登录并进入 `/pages/elder/home`
- [ ] 守护人端账号可成功登录并进入 `/pages/guardian/home`
- [ ] 登录失败时可看到表单错误提示
- [ ] 退出登录后会回到登录页并清空当前角色信息

## 聊天与风险识别

- [ ] 老年端进入聊天会话页时可看到加载态
- [ ] 弱网或断网时，会话页会展示弱网提示和重试按钮
- [ ] 发送包含“转账 / 验证码 / 链接”等关键词的消息后，可触发风险提示
- [ ] 守护人端风险通知页可手动刷新风险结果
- [ ] 风险通知为空时可看到空状态文案

## 联系人与老人列表

- [ ] 联系人列表为空时展示空状态
- [ ] 老人列表为空时展示空状态
- [ ] 从联系人页进入聊天和通话页正常
- [ ] 从老人列表进入聊天回访和语音回访正常

## 求助与通话

- [ ] 老年端一键求助后可进入成功反馈页
- [ ] 守护人端首页能看到新的求助提醒
- [ ] 守护人可将求助标记为处理中
- [ ] 语音通话拨打、接听、挂断后能生成通话记录
