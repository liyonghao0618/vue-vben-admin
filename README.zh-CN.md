# 桑榆智盾：AI反电诈守护系统

桑榆智盾：AI反电诈守护系统，是一个面向老年人反电诈场景的比赛项目，围绕老年用户、子女、社区工作人员、系统管理员四类角色，提供风险识别、风险提醒、远程监护、社区处置和后台配置能力。

## 项目定位

- 目标问题：降低老年人遭遇电信诈骗时的识别成本和求助门槛
- 核心思路：让老年人、子女、社区三方形成协同闭环
- 当前阶段：V1 比赛演示版，重点保证核心链路可跑通、页面语义清晰、演示稳定

## 当前能力

- 老年端：风险提醒、一键求助、亲属绑定、防骗知识、适老设置
- 子女端：监护总览、老人列表、风险详情、通知记录、远程提醒
- 社区端：辖区总览、重点老人、风险工单、宣教管理、统计报表
- 管理后台：用户管理、角色权限、风险规则、内容管理、系统配置
- 后端能力：认证、绑定关系、风险识别、告警生成、通知联动、工单流转、配置管理

## 推荐演示链路

`登录 -> 风险识别 -> 风险告警 -> 子女通知 -> 社区工单 -> 管理配置`

## 本地运行

1. 安装依赖

```bash
pnpm install
```

2. 启动前端

```bash
pnpm dev:antd
```

3. 启动后端

```bash
cd apps/backend-fastapi
source .venv/bin/activate
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

4. 打包前端

```bash
pnpm build:antd
```
